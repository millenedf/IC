Introdução

Bem-vindo! Esta página explora conceitos fascinantes de sistemas dinâmicos, com foco na roda de Malkus, um sistema caótico que ajuda a entender como padrões complexos emergem de equações aparentemente simples.

O que é a Roda de Malkus?

A roda de Malkus é um dispositivo mecânico que ilustra comportamentos caóticos. Ela consiste em copos presos a uma roda rotativa, com água fluindo entre eles. O movimento é descrito por um sistema de equações diferenciais que modela o equilíbrio e a dinâmica da água nos copos.

Figura: Representação da roda de Malkus.

Sistemas Dinâmicos

Sistemas dinâmicos são modelos matemáticos que descrevem como o estado de um sistema evolui ao longo do tempo. Exemplos incluem o sistema de Lorenz e a roda de Malkus, que exibem comportamentos caóticos sensíveis às condições iniciais.

// Exemplo de equações do sistema de Lorenz
sigma = 10;
rho = 28;
beta = 8/3;

x' = sigma * (y - x);
y' = x * (rho - z) - y;
z' = x * y - beta * z;

Simulação Computacional

Utilizamos métodos numéricos, como o método de Runge-Kutta de quarta ordem, para resolver as equações diferenciais da roda de Malkus. Abaixo está um exemplo de código em Python para simulação:

import numpy as np
from scipy.integrate import solve_ivp

def roda_malkus(t, y, L, k):
    theta, omega = y
    dtheta_dt = omega
    domega_dt = -np.exp(- (theta / (L/2)) ** k) * omega
    return [dtheta_dt, domega_dt]

sol = solve_ivp(roda_malkus, [0, 50], [0.1, 0.0], args=(1.0, 4))
